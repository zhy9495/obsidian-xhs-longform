# Contributing

Bug reports and focused pull requests are welcome.

## Development

1. Install Node.js 24 or later.
2. Run `npm ci`.
3. Run `npm run lint`, `npm test`, and `npm run build` before opening a pull
   request.

Do not commit `main.js`; production artifacts belong in GitHub releases. Do
not add proprietary fonts or font files without a verified redistribution
license and the corresponding license text.
