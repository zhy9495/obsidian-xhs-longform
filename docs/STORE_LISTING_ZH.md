# 插件市场中文介绍

## 一句话介绍

把 Obsidian Markdown 长文一键排成适合小红书发布的 3:4 多图，支持实时预览、自动分页、封面图、头像、字体、配色和页码。

## 简短介绍

XHS Longform Exporter 是一个面向小红书长图文的 Obsidian 导出插件。打开笔记后，可以在左侧调整封面图、头像、标题、字体、字号、配色和边距，并在右侧实时查看最终效果。插件会根据真实字体和页面空间自动分页，确保正文不会与底部昵称、页码互相遮挡，最后一次导出全部 PNG。

所有排版和图片生成都在本地完成。插件不会把笔记、头像、封面图或自定义字体上传到开发者服务器。

## 使用方法

1. 打开一篇 Markdown 笔记。
2. 在命令面板运行“导出当前笔记为小红书图片”，或右键笔记文件选择“导出为小红书图片”。
3. 在左侧调整设置，在右侧检查实时预览。
4. 点击“导出 PNG”，得到连续编号的 3:4 图片。

## 核心功能

- 自动把长文分页为 1080×1440 PNG。
- 封面图、头像和文件名标题可以独立开关。
- 支持头像昵称、15 字以内的自定义文字、固定底部署名和页码。
- 标题、小标题和正文可以分别调整字号。
- 支持本地图片、列表、引用、表格、高亮和行内代码。
- 支持 PingFang SC、本机字体、按需下载的 OFL 字体和用户导入字体。
- 全程本地处理，不上传笔记内容。
